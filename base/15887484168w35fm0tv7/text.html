GitHub Gist — это обычные git-репозитории, которые просто иначе отображаются в интерфейсе GitHub. Единственное заметное отличие от полноценных репозиториев — в Gist нельзя использовать директории, все файлы должны находиться в корне репозитория.


Чтобы конвертировать обычный репозиторий в Gist с сохранением истории файлов, можно сделать следующее:


1. Создать новый пустой Gist;

2. Склонировать его локально:


git clone git@gist.github.com:792bxxxxxxxxxxxxxxx9.git


3. Войти в эту директорию:

4. Скачать и слить изменения из другого репозитория:


git pull git@github.com:<user>/<repo>.git


5. Запушить всё на GitHub Gist:


git push
