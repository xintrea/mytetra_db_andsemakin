Обычная установка:


pip install black


Через pipenv:


pipenv install --dev --pre black


Флаг --pre нужен, потому что на начало 2020 года у black еще не было ни одного стабильного релиза, только альфа-версии. Это имеет побочный эффект, потому что тогда все остальные пакеты через pipenv начинают тоже прилетать пререлизные. Ох уж этот блэк. Баг pipenv: https://github.com/pypa/pipenv/issues/1760
