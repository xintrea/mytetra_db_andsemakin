Чтобы подключить к сайту на GitHub Pages собственный домен нужно в корне репозитория создать файл CNAME с доменом, например, вот так:


semakin.dev


Рекомендую зайти в настройки репозитория на GitHub и убедиться, что доменное имя распознано, а также выставить галочку "Enforce HTTPS".


После этого в настройках домена (обычно это делается через регистратора) нужно создать записи типа A указывающие на сервера GitHub Pages (актуальный список в официальной документации):


185.199.108.153

185.199.109.153

185.199.110.153

185.199.111.153


Через какое-то время сайт заработает по новому адресу. GitHub Pages даже автоматически сгенерирует SSL-сертификат для нового имени!
