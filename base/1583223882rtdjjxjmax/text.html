Установка pypy:


asdf list all python | grep pypy

asdf install python pypy3.6-7.3.0


Настройка:


После установки возможна следующая ошибка:

$ pypy3

asdf: No version set for command pypy3

you might want to add one of the following in your .tool-versions file:


Для того, чтобы заработала команда pypy2 или pypy3, установленная через asdf, нужно выбрать текущую версию, например, так:


asdf global python 3.8.2 pypy3.6-7.3.0 pypy2.7-7.3.0
