Иногда ни с того, ни с сего при попытке взаимодействовать с удаленным сервером возникает ошибка:


ERROR: Repository not found.

fatal: Could not read from remote repository.


Please make sure you have the correct access rights

and the repository exists.


Скорее всего, это означает, что нужные SSH ключи по какой-то причине пропали из ssh-agent, и нужно их вернуть обратно:


ssh-add ~/.ssh/id_rsa
