1. Открыть cmd с правами администратора;

2. Найти интересующий пакет на сайте: https://chocolatey.org/packages;

3. Выполнить команду по установке:


choco install sysinternals


или сразу несколько программ:


choco install notepadplusplus googlechrome 7zip


Чтобы избежать диалогов (например, принятие лицензий), можно добавить к команде флаг -y:


choco install -y notepadplusplus googlechrome 7zip
