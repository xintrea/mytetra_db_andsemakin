Многие программы, например, Etcher или UNetBootin не способны корректно создать загрузочный диск с Windows, хотя отлично подходят для записи образов Linux. Для Windows есть специализированная программа -- WoeUSB.


Установка:


Ubuntu:


sudo add-apt-repository ppa:nilarimogard/webupd8

sudo apt-get update

sudo apt-get install woeusb


Fedora:


sudo dnf install woeusb


Использование:


1. Форматировать флешку в NTFS;

2. Отмонтировать тома от системы:

* в GNOME кнопка Eject в файловом менеджере извлекает девайс целиком, можно через приложение Disks отмонтировать только конкретный том при помощи кнопки Stop;

3. Запустить WoeUSB в графическом режиме через меню ОС;

4. Выбрать образ, выбрать целевое устройство, начать запись.


Оригинальные образы Windows можно скачать отсюда: https://www.microsoft.com/en-us/software-download/windows10ISO

FAQ по WoeUSB: https://github.com/slacka/WoeUSB/wiki/F.A.Q.
