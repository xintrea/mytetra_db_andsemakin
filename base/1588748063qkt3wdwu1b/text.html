Чтобы встроить код только из одного файла из многофайлового gist, можно использовать вот такой приём:


1. Скопировать код для встраивания в интерфейсе gist;

2. Добавить в конец URL ?file=your_file.py, например, вот так:


<script src="https://gist.github.com/4505639.js?file=macroBuild.scala" type="text/javascript"></script>
